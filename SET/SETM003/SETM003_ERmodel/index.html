01_SETM003(ACMM003)會計科目維護_ER-Model

在 “SETM003(ACMM003)會計科目維護.doc” 文件中，涉及的主要資料表為 GL_ACCNT_MASTER，並且它也涉及到表 MIS_SYST，用於獲取子系統的名稱。以下是這些資料表的欄位和對應的中文備註。

1. GL_ACCNT_MASTER（會計科目檔）

欄位名稱 資料型態 中文備註
ACCNT_NO VARCHAR2(10) 科目代號
ACCNT_KIND VARCHAR2(1) 科目類別
ACCNT_NAME VARCHAR2(30) 科目名稱
BUDGET_CODE VARCHAR2(1) 預算科目
ACCNT_DESC VARCHAR2(60) 主要說明
CREATE_OFFSET_CODE VARCHAR2(1) 立沖科目
SUBSYSTEM_NO VARCHAR2(3) 子系統代號
ACCNT_D_C VARCHAR2(1) 借/貸
ASSEST_ACCNT_NO VARCHAR2(1) 資產科目
CRE_DATE DATE 開始日
END_DATE DATE 結束日
SPREAD_ACCNT_NO VARCHAR2(1) 衍生科目
LAST_UPDATE_DATE DATE 最近維護日

2. MIS_SYST（子系統檔）

欄位名稱 資料型態 中文備註
SYST_IDEN VARCHAR2 子系統代號
SYST_NAME VARCHAR2 子系統名稱

這些欄位都是文件中提到的 SETM003Controller API 相關的部分。在文件中描述了各種作業流程，例如查詢、編輯和檢核科目代號等操作【277:1†source】.
