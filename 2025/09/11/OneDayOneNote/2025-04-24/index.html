2025-04-24-Docker Desktop 最新版本 (2025年) 初學者建議設定清單

好 👍 我幫你整理一份 Docker Desktop 最新版本 (2025年) 初學者建議設定清單，避免踩雷，讓你先能順利用 Docker。

🛠 初學者建議 Docker Desktop Settings

1. General

✅ 建議勾選：

  • ✔ Start Docker Desktop when you sign in → 讓你開機自動啟動（方便）。
  • ✔ Use the WSL 2 based engine → Windows 必須勾，效能更好。
  • ✔ Show CLI Hints → 初學者友好，會提示一些指令用法。
  • ✔ Enable Scout image analysis → 可以幫你檢查 image 是否安全。

⚪ 建議不要勾：

  • ⚪ Expose daemon on tcp://localhost:2375 without TLS → 有安全風險，除非你有舊程式需要。
  • ⚪ Send usage statistics → 看個人，想保護隱私就不要勾。
  • ⚪ Use Enhanced Container Isolation → 要付費的 Business 版才有，不需要。

3. Builders

4. Resources (CPU, Memory, Disk)

  • 初學者建議保持 預設值。
  • 如果你跑比較大的服務（像 MySQL、AI 模型），記得適度調高 Memory（例如 4GB → 8GB）。

5. AI (Beta)

  • ⚪ Enable Docker Model Runner → 初學者不用勾，因為你暫時不跑 AI 模型。
  • ⚪ Enable host-side TCP support → 同上，不需要。

6. Kubernetes (如果有看到)

  • 初學者建議 不要先開。
  • Kubernetes 適合多人系統與生產環境，你目前學 Docker 基本用法就好。

✅ 總結 — 初學者最佳設定

  • General → 只勾 Start with Windows / WSL2 / CLI Hints / Scout
  • Docker Engine → 預設即可
  • Builders → 保持 default，不用調
  • Resources → 預設即可
  • AI → 全部關閉
  • Kubernetes → 關閉
