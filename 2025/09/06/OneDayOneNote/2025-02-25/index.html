🔍 prototype 是什麼？為什麼要看？

在 JavaScript 中，所有物件都有原型（prototype）。這個「原型」是物件的隱藏屬性，用來繼承其他物件的屬性與方法。

簡單來說：

prototype 是 JavaScript 的繼承機制，是「物件的爸爸」或「模板」。
