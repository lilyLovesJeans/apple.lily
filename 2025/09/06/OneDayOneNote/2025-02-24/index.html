🎯 小整理：箭頭函式的隱式 return

語法 說明
x => x * 2 ✅ 隱式 return x * 2
x => { return x * 2 } ✅ 明確 return
x => { x * 2 } ⛔ 沒有 return，會是 undefined

✅ 正確理解你看到的寫法

這是完全正確的語法，只要你 return 的東西是單行表達式（expression），就可以省略大括號和 return，語法更簡潔。
