按鈕顏色沒有「絕對規定」，但大部分 UI 設計（像 Material Design、Bootstrap、Ant Design、Quasar 等框架）都有一些「常見的配色習慣」：

🔑 常見按鈕顏色建議

1. 登入 / 確認 / 儲存

  • ✅ 主要動作（Primary Action）

  • 常見顏色：藍色 (Blue) 或 綠色 (Green)

  • 例子：

    • 藍色：#1976d2、#409eff（Element UI）、Bootstrap 的 .btn-primary
    • 綠色：#28a745（Bootstrap Success）

👉 因為「藍色」給人穩定、信任的感覺，
👉 「綠色」則表示成功、完成。

2. 刪除 / 危險動作

  • ⚠️ Danger / Destructive Action

  • 常見顏色：紅色 (Red)

  • 例子：

    • #e74c3c（常見 danger 紅色）
    • Bootstrap .btn-danger
    • Quasar color="negative"

👉 紅色能立即讓使用者警覺「這個動作不可逆」。

3. 新增 / 建立

  • ➕ Add / Create

  • 常見顏色：綠色 (Green)

  • 例子：

    • #4caf50（Material Design Green 500）
    • Bootstrap .btn-success

👉 綠色象徵「新增、成長、正向」。

4. 取消 / 關閉

  • ❌ Secondary / Neutral

  • 常見顏色：灰色 (Gray)

  • 例子：

    • #6c757d（Bootstrap .btn-secondary）
    • Material Design「Cancel」多用灰色文字 + 輕量按鈕

👉 灰色代表「不強調、不會造成危險」。
