✅ 所以 !window.L 的意思是：

  • 「如果 window.L 不存在」（也就是 Leaflet 沒有正確載入）
  • 就顯示錯誤或中止程式

這樣做的目的是避免後面用 L.map(...) 時報錯 L is not defined。

✅ 補充：為什麼我們不用 import L from 'leaflet'？

因為你選擇了 CDN 模式（不用 npm 安裝），所以：

  • 我們不寫 import
  • 而是用 <script src="..."> 載入 Leaflet
  • 這會在全域自動建立 window.L

這種做法常見於：

  • 練習、教學
  • 無需打包的專案（例如 Replit、CodePen）
