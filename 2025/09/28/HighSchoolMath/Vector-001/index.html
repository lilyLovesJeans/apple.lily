第12題：外心問題, 設O為△ABC之外心，已知AB=6，BC=2√7，AC=4，且AO = r AB + s AC

發表於 2025-09-28 分類於 高中數學 閱讀次數：
第12題：外心問題, 設O為△ABC之外心，已知AB=6，BC=2√7，AC=4，且AO = r AB + s AC

第12題：外心問題, 設O為△ABC之外心，已知AB=6，BC=2√7，AC=4，且AO = r AB + s AC

快速答案

  1. (\displaystyle \vec{AB}\cdot\vec{AC}=12.)
  2. (\displaystyle (r,s)=\Bigl(\tfrac{4}{9},\tfrac{1}{6}\Bigr).)

怎麼想

想像 A、B、C 是三根柱子立在操場：

  • 我們先要看從 A 指向 B 的箭（AB）跟從 A 指向 C 的箭（AC）有沒有差不多同方向（這就是「點積」的意義）。
  • 再找一個點 O，是可以用同一個圈套住三根柱子的圓心（外心）。問：從 A 走到 O，能不能先走某個倍數的 AB，再走某個倍數的 AC？求那兩個倍數 (r,s)。

大方向：把 A 放原點、把 AB 放 x 軸，算點積（或角），求外心 O 的座標，再解線性方程求 (r,s)。

已知（把問題變成數字）

我們採用座標：
[
A=(0,0),\quad B=(6,0),\quad C=(2,,2\sqrt3).
]
因此
[
\vec{AB}=(6,0),\qquad \vec{AC}=(2,2\sqrt3).
]
題目給三邊長也為： (AB=6,\ AC=4,\ BC=2\sqrt7)（座標與邊長互相吻合，可用距離公式檢查）。

Step 1 — 計算點積 (\vec{AB}\cdot\vec{AC}) （圖：triangle_step1.png / triangle_step2.png）

方法 A（用座標成分）
[
\vec{AB}\cdot\vec{AC}=6\cdot2 + 0\cdot(2\sqrt3)=12.
]

方法 B（用長與角）（可互相驗算）
先用餘弦定理求 (\cos A)：
[
\cos A=\frac{AB^2+AC^2-BC^2}{2\cdot AB\cdot AC}
=\frac{36+16-28}{2\cdot6\cdot4}=\frac{24}{48}=\tfrac12.
]
所以 (A=60^\circ)。
再用點積公式：
[
\vec{AB}\cdot\vec{AC}=|AB||AC|\cos A=6\cdot4\cdot\frac12=12.
]

結論（第一題）：(\boxed{\vec{AB}\cdot\vec{AC}=12}).

Step 2 — 找外心 (O)（外接圓圓心）（圖：triangle_step3.png 與 合併圖 triangle_circumcenter.png）

理論：外心是三角形三邊中垂線的交點（也就是三個頂點到外心距離都一樣，外接圓心）。

因為 AB 在 x 軸上：

  • AB 的中點：((3,0))，中垂線就是直線 (x=3)。
  • AC 的中點：((1,\sqrt3))。
    AC 的斜率為 (\sqrt3)，中垂線的斜率為負倒數 (-\tfrac{1}{\sqrt3})，經過 ((1,\sqrt3))，方程：
    [
    y-\sqrt3 = -\frac{1}{\sqrt3}(x-1).
    ]
    把 (x=3) 代入，得到
    [
    y=\sqrt3 - \frac{2}{\sqrt3} = \frac{1}{\sqrt3}.
    ]
    因此外心
    [
    O=\Bigl(3,\ \frac{1}{\sqrt3}\Bigr).
    ]

向量表示：
[
\vec{AO}=O-A=\Bigl(3,\ \tfrac{1}{\sqrt3}\Bigr).
]

Step 3 — 把 (\vec{AO}) 寫成 (r\vec{AB}+s\vec{AC})（圖：triangle_step4.png）

目標：找 (r,s) 使得
[
\vec{AO}=r\vec{AB}+s\vec{AC}.
]
代入向量成分：
[
r(6,0)+s(2,2\sqrt3)=(3,\ \tfrac{1}{\sqrt3}).
]
把 x、y 分別寫出方程：
[

]

先從第二式解 (s)：
[
2\sqrt3\ s = \frac{1}{\sqrt3}\quad\Rightarrow\quad s=\frac{1}{6}.
]
再代回第一式解 (r)：
[
6r + 2\cdot\frac{1}{6} = 3
\quad\Rightarrow\quad 6r + \frac{1}{3} = 3
\quad\Rightarrow\quad 6r = \frac{8}{3}
\quad\Rightarrow\quad r = \frac{4}{9}.
]

結論（第二題）：(\boxed{(r,s)=\Bigl(\tfrac{4}{9},\tfrac{1}{6}\Bigr)}).

最後的簡單生活比喻（小學生回顧）

  • (\vec{AB}) = 從 A 向右走 6 步。
  • (\vec{AC}) = 從 A 斜上走 4 步（方向 60°）。
  • (r=\tfrac{4}{9}) 表示「走 AB 的 4/9 倍」（也就是 6×4/9 = 8/3 步）。
  • (s=\tfrac{1}{6}) 表示「走 AC 的 1/6 倍」（也就是 4×1/6 = 2/3 步）。
    把這兩段頭對尾接起來，就恰好從 A 走到外心 O。

這題會在哪些情況用到？

  • 設計圓形花園（想要圓心讓三個點在圓上）→ 要找外心。
  • 測量與定位、工程圖（外接圓、等距性）→ 利用中垂線、外心概念。
  • 向量分解概念是導航和機器人路徑規劃常用的技巧（把移動拆成幾個方向的組合）。
