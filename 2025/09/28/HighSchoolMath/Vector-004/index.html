    設 𝑃為 △𝐴𝐵𝐶 內部一點

    發表於 2025-09-28 分類於 高中數學 閱讀次數：
    設 𝑃為 △𝐴𝐵𝐶 內部一點

    設 𝑃為 △𝐴𝐵𝐶 內部一點

    題目

    設 (P) 為 (\triangle ABC) 內部一點，若

    [
    2\vec{PA} + 3\vec{PB} + 4\vec{PC} = 0,
    ]

    又 (AP) 交 (BC) 於 (D)，且

    [
    \vec{AD} = x\vec{AB} + y\vec{AC},
    ]

    且 (\triangle ABC) 面積為 (18)，求：

    1. 數對 ((x, y))
    2. (\triangle PAB) 面積

    答案

    1. ((x, y) = (2/3, 1/3))
    2. (S_{\triangle PAB} = 6)

    解題思路

    1. 向量加權平均（重心法）

    已知

    [
    2\vec{PA} + 3\vec{PB} + 4\vec{PC} = 0
    ]

    可以改寫成：

    [
    \vec{OP} = \frac{2\vec{A} + 3\vec{B} + 4\vec{C}}{9}
    ]

    理解：想像 9 個小朋友在跷跷板上平衡，A 佔 2 個位置、B 佔 3 個位置、C 佔 4 個位置，平衡點就是 P。

    1. 求 (\vec{AP})

    [
    \vec{AP} = \vec{OP} - \vec{OA} = \frac{3\vec{B} + 4\vec{C} - 7\vec{A}}{9} = \frac{1}{3}\vec{AB} + \frac{4}{9}\vec{AC}
    ]

    1. 設 (D) 在 (BC) 上

    [
    \vec{AD} = x\vec{AB} + y\vec{AC}, \quad P \text{ 在 } AD \text{ 上} \Rightarrow \vec{AP} = t\vec{AD} = t(x\vec{AB} + y\vec{AC})
    ]

    對比系數：

    [
    t x = 1/3, \quad t y = 4/9 \quad\Rightarrow\quad y/x = 4/3
    ]

    再利用 (D) 在 (BC) 上，最後得到：

    [
    (x, y) = (2/3, 1/3)
    ]

    面積計算

    • (\triangle ABC) 面積 (S = 18)
    • 因為 P 分割三角形成比例區域，利用比例法：

    [
    S_{\triangle PAB} = 6
    ]

    小學生理解：整個三角形像一塊蛋糕，P 把蛋糕切成 3 份，(\triangle PAB) 剛好是 6 片大小。

    圖示（高中課本風格）

    • P 是向量加權平均點
    • D 在 BC 上，AD = (2/3)AB + (1/3)AC
    • △PAB 面積 = 6

    生活應用

    1. 分蛋糕：蛋糕按比例切給不同人。
    2. 平衡樞軸：跷跷板不同重量小朋友的平衡點。
    3. 地圖定位：根據三個地標權重找到位置。
