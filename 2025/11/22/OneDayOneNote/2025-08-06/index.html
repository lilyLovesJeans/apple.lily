    為什麼在 Lenovo Windows 電腦 要裝 Git Bash？

    ✅ 什麼是 Git Bash？

    Git Bash 是 Windows 裡的一個「模擬 Linux/Mac 終端機」的工具。

    它隨著 Git for Windows 一起安裝，裡面包含：

    • Git 指令（如 git clone / git status）
    • Bash 終端機（Linux/Mac 常用）
    • Linux 常用工具（如 ls, pwd, mv, grep…）

    你可以把它當成：

    👉「在 Windows 裡提供一個像 Mac/Linux 的指令環境」。

    ✅ 為什麼 Lenovo（或任何 Windows 電腦）需要用 Git Bash？

    因為 Windows 的預設 cmd 或 PowerShell 跟 Linux/Mac 指令不一樣。

    例如：

    功能 Windows CMD Linux/Mac（也就是 Git Bash）
    查看資料夾 dir ls
    移動資料夾 cd .. cd ..
    列印目前路徑 cd pwd
    移動檔案 move mv

    你在寫程式時，教學、範例、YouTube、GitHub 都會用 Linux/Mac 的指令格式。

    所以如果你用 Windows：

    👉 用 Git Bash，可以讓你跟著教學使用一模一樣的指令。

    🎯 更重要的理由 —— 開發者全部都用 Bash

    無論你從事：

    • 前端（Vue、React）
    • 後端（Node.js / NestJS）
    • Docker
    • AWS、GCP、Azure
    • Python
    • DevOps

    👉 教學 99% 都會使用 Bash 指令，而不是 CMD 或 PowerShell。

    所以開發者慣例：

    Windows + Git Bash = 最貼近真正工程師的環境

    ✨小結（你記這三句就好）

    1. Git Bash 是 Linux/Mac 的指令環境，在 Windows 使用。
    2. 它讓你在 Windows 上使用所有開發者常用的指令（ls/mv/pwd/git）。
    3. 它跟教學環境一致，學習更輕鬆、不會被 Windows 指令卡住。
