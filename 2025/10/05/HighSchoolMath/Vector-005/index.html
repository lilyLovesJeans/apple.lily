    Vector-005 在△ABC中,AD交BCD點,BE交AD於E點,且∠ACB = 30°,∠EDB = 60°,∠AEB =120°。若CD=15,ED=7,則AB=

    結論：**(\overline{AB}=13)**。$

    最後的綜合圖：

    步驟 1：座標化 + 利用已知角度找 (A)（圖 1）

    想法（為什麼要這樣做）：把 (C) 放在原點，將 (BC) 放在 (x)-軸上，可以把角度轉成直線斜率或方程式，使幾何問題變成解代數方程，計算方便且系統化。
    $

    • 設
      [
      C=(0,0),\qquad D=(15,0)
      ]
      （因為題目給 (\overline{CD}=15)，且 (D) 在 (BC) 上，我把 (C) 放在原點， (BC) 在 (x)-軸上）
    • 由 (\angle ACB=30^\circ)，所以直線 (CA) 的方向角為 (30^\circ)，其斜率為
      [
      \tan 30^\circ=\frac{1}{\sqrt3}.
      ]
      因此直線 (CA) 的方程為
      [
      y=\frac{1}{\sqrt3},x. \tag{1}
      ]
      $
    • 由 (\angle EDB=60^\circ) 且 (E) 在 (AD) 上（注意：(\angle EDB) 其實是 (AD) 與 (DB) 的夾角），可得 (AD) 通過 (D(15,0))，且方向角為 (60^\circ)。所以直線 (AD) 的方程為
      [
      y=\sqrt3,(x-15). \tag{2}
      ]
    • 將 (1) 與 (2) 聯立求交點（即 (A)）：
      [
      \frac{1}{\sqrt3}x=\sqrt3(x-15)\quad\Longrightarrow\quad x=3(x-15)\quad\Longrightarrow\quad 2x=45
      ]
      [
      \Rightarrow\ x=\frac{45}{2}=22.5.
      ]
      再代回 (1)：
      [
      y=\frac{1}{\sqrt3}\cdot\frac{45}{2}=\frac{45}{2\sqrt3}=\frac{15\sqrt3}{2}.
      ]
    • 所以
      [
      A=\Big(\tfrac{45}{2},\ \tfrac{15\sqrt3}{2}\Big).
      ]

    （圖 1：顯示 (C) 為原點，從 (C) 作 (30^\circ) 射線、從 (D) 作 (60^\circ) 射線，二者交點為 (A)。）

    補充計算（檢查長度）：
    [
    AC=\sqrt{\Big(\tfrac{45}{2}\Big)^2+\Big(\tfrac{15\sqrt3}{2}\Big)^2}
    =\tfrac{1}{2}\sqrt{45^2+15^2\cdot3}
    =\tfrac{1}{2}\sqrt{2700}=\tfrac{1}{2}\cdot30\sqrt3=15\sqrt3.
    ]

    步驟 2：利用 (AD) 的方向找 (E)（圖 2）

    已知： (ED=7)，且 (E) 在 (AD) 上，且我們已知 (AD) 的方向是 (60^\circ)。
    為什麼這樣做：沿已知方向走固定距離即可得到 (E) 的座標。

    • 由前步驟 (A) 與 (D) 的座標，可求得 (AD) 長度（檢查）：
      [
      AD=\sqrt{\Big(\tfrac{45}{2}-15\Big)^2+\Big(\tfrac{15\sqrt3}{2}-0\Big)^2}
      =\sqrt{\Big(\tfrac{15}{2}\Big)^2+\Big(\tfrac{15\sqrt3}{2}\Big)^2}
      =\tfrac{15}{2}\sqrt{1+3}=15.
      ]
    • 因為 (AD=15)，且 (ED=7)，所以 (AE=AD-ED=15-7=8)（數值檢查一致）。
    • (AD) 的單位方向向量是 ((\cos60^\circ,\sin60^\circ)=(\tfrac12,\tfrac{\sqrt3}{2}))。由 (D) 往上量 (7) 得到 (E)：
      [
      E=D+7\big(\tfrac12,\tfrac{\sqrt3}{2}\big)=\Big(15+\tfrac{7}{2},\ \tfrac{7\sqrt3}{2}\Big)=\Big(\tfrac{37}{2},\ \tfrac{7\sqrt3}{2}\Big).
      ]
    • 檢查 (AE) 向量（從 (E) 指向 (A)）：
      [
      \overrightarrow{EA}=A-E=\Big(\tfrac{45}{2}-\tfrac{37}{2},\ \tfrac{15\sqrt3}{2}-\tfrac{7\sqrt3}{2}\Big)=(4,\ 4\sqrt3),
      ]
      長度 (|\overrightarrow{EA}|=\sqrt{4^2+(4\sqrt3)^2}=\sqrt{16+48}= \sqrt{64}=8)，與上面 (AE=8) 相符。

    （圖 2：顯示 (A,D,E) 在同一直線上，標明 (AD=15,,ED=7,,AE=8)。）

    步驟 3：以 (\angle AEB=120^\circ) 求 (B)（圖 3）

    為什麼採用向量/內積：角度條件 (\angle AEB=120^\circ) 可以用「兩向量的內積（點積）」公式轉成代數方程，容易代數求解出 (B) 的 (x)-座標（因為我們已把 (B) 放在 (x)-軸上）。

    • 設 (B=(b,0))。則
      [
      \overrightarrow{EA}=(4,\ 4\sqrt3),\qquad \overrightarrow{EB}=\big(b-\tfrac{37}{2},\ -\tfrac{7\sqrt3}{2}\big).
      ]
    • 兩向量的內積（dot product）公式（高中應會見到）：
      [
      \overrightarrow{u}\cdot\overrightarrow{v}=|\overrightarrow{u}|,|\overrightarrow{v}|\cos\theta,
      ]
      其中 (\theta) 為兩向量夾角。應用在這裡，(\theta=120^\circ,\ \cos120^\circ=-\tfrac12)。
    • 先計算內積的代數式：
      [
      \overrightarrow{EA}\cdot\overrightarrow{EB}
      =4\cdot\frac{2b-37}{2}+(4\sqrt3)\cdot\Big(-\frac{7\sqrt3}{2}\Big)
      =2(2b-37)-42
      =4b-116.
      ]
      （中間項：((4\sqrt3)(-7\sqrt3/2)=-28\cdot\frac{3}{2}=-42)。）
    • 兩向量的長度：
      [
      |\overrightarrow{EA}|=8,\qquad
      |\overrightarrow{EB}|=\tfrac{1}{2}\sqrt{(2b-37)^2+147}.
      ]
      （因為平方和：((-\tfrac{7\sqrt3}{2})^2=\tfrac{147}{4})。）
    • 內積等式變成
      [
      \overrightarrow{EA}\cdot\overrightarrow{EB}=|\overrightarrow{EA}|,|\overrightarrow{EB}|\cos120^\circ
      ]
      即
      [
      4b-116=8\cdot\frac{1}{2}\sqrt{(2b-37)^2+147}\cdot\big(-\tfrac12\big).
      ]
      簡化：
      [
      \frac{4b-116}{4}=-\frac{1}{2}\sqrt{(2b-37)^2+147}
      \quad\Longrightarrow\quad 4b-116=-2\sqrt{(2b-37)^2+147}.
      ]
      兩邊同乘 (-1)，得
      [
      116-4b=2\sqrt{(2b-37)^2+147}.
      ]
    • 把等式平方（注意：平方會引入可能的符號，但由圖形的方向與座標可以排除負根）：
      [
      (116-4b)^2=4\big((2b-37)^2+147\big).
      ]
      展開整理（逐步計算）：
      [
      (116-4b)^2=16b^2-928b+13456,
      ]
      [
      4\big((2b-37)^2+147\big)=4(4b^2-148b+1369+147)=16b^2-592b+6064.
      ]
      等式兩邊相等得到：
      [
      16b^2-928b+13456=16b^2-592b+6064.
      ]
      消去 (16b^2)，整理：
      [
      -928b+13456=-592b+6064 \quad\Longrightarrow\quad -336b+7392=0,
      ]
      [
      \Rightarrow b=\frac{7392}{336}=22.
      ]
    • 因此 (B=(22,0))。所以
      [
      BC=b=22,\qquad BD=BC-CD=22-15=7.
      ]
      （這裡會發現 (BD=7=ED) — 幾何上是一個漂亮的對稱數值結果。）

    （圖 3：畫出 (E) 與 (B(22,0)) 連線，並示 (\angle AEB=120^\circ) 的位置說明。）

    步驟 4：計算 (\overline{AB})（圖 4）

    • 現在有 (A=\big(\tfrac{45}{2},\tfrac{15\sqrt3}{2}\big)) 與 (B=(22,0))。計算距離：
      [
      AB^2=\Big(\tfrac{45}{2}-22\Big)^2+\Big(\tfrac{15\sqrt3}{2}-0\Big)^2
      =\Big(\tfrac{1}{2}\Big)^2+\Big(\tfrac{15\sqrt3}{2}\Big)^2
      =\tfrac{1}{4}+\tfrac{675}{4}=\tfrac{676}{4}=169.
      ]
      [
      \Rightarrow AB=\sqrt{169}=13.
      ]

    （圖 4：完整三角形 (ABC)，標示 (AB=13)）

    總結（包括向量與三角函數的簡單說明）

    最終答案：(\boxed{AB=13})。

    解題思路（快速回顧）：

    1. 用座標化（把 (C) 設原點、 (BC) 放 (x)-軸）把角度條件轉成直線方程（使用斜率 (\tan)），找到 (A)。
    2. 用 (AD) 的方向與已知 (ED) 的長度定位 (E)。
    3. 用角度條件 (\angle AEB=120^\circ) 把「角度」轉成「向量內積（dot product）」的代數式，解出 (B) 的 (x)-座標。
    4. 最後計算兩點距離得 (AB)。

    向量（簡單概念）：

    • 向量就是有大小與方向的量，在平面常用 ((x,y)) 來表示；兩點 (P(x_1,y_1)) 與 (Q(x_2,y_2)) 的向量 (\overrightarrow{PQ}=(x_2-x_1,\ y_2-y_1))。
    • 兩向量 (\vec u=(u_x,u_y)) 與 (\vec v=(v_x,v_y)) 的內積：
      [
      \vec u\cdot\vec v=u_xv_x+u_yv_y,
      ]
      同時有
      [
      \vec u\cdot\vec v=|\vec u|,|\vec v|\cos\theta,
      ]
      其中 (\theta) 是兩向量的夾角。這條公式常用於把「角度」變成「代數方程」，是本題求 (B) 的關鍵。

    三角函數（簡單概念）：

    • 在直角三角形中：
      [
      \sin\theta=\frac{\text{對邊}}{\text{斜邊}},\qquad
      \cos\theta=\frac{\text{鄰邊}}{\text{斜邊}},\qquad
      \tan\theta=\frac{\sin\theta}{\cos\theta}=\frac{\text{對邊}}{\text{鄰邊}}.
      ]
    • 在座標上，角度 (\theta) 的方向向量可以寫成 ((\cos\theta,\sin\theta))，因此知道直線與 (x)-軸的角度就能給出方向向量與斜率（(\tan\theta)），常用來寫出直線方程。
