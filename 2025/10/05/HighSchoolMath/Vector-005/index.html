    Vector-005 在△ABC中，AD 交 BC 於 D，BE 交 AD 於 E，且 ∠ACB = 30°, ∠EDB = 60°, ∠AEB =120°。若 CD=15,ED=7,則 AB = ?

    題目重述（Vector-005）

    在 △ABC 中，AD 交 BC 於 D，BE 交 AD 於 E，且
    [
    \angle ACB = 30^\circ,\quad \angle EDB = 60^\circ,\quad \angle AEB = 120^\circ.
    ]
    若 (CD = 15,; ED = 7)，求 (AB)。

    結論：(\overline{AB}=13)。

    最後的綜合圖：

    步驟 1：座標化並利用已知角度求 (A)（圖 1）

    想法（為什麼要這樣做）：把 (C) 放在原點，將 (BC) 放在 (x)-軸上，可以把角度轉成直線斜率或方程式，使幾何問題變成代數方程，計算較方便。

    • 設
      [
      C=(0,0),\qquad D=(15,0),
      ]
      因為題目給 (CD=15)，且 (D) 在 (BC) 上，所以把 (C) 放在原點、(BC) 放 (x)-軸。
    • 由 (\angle ACB=30^\circ)，直線 (CA) 與 (x)-軸的夾角為 (30^\circ)，其斜率為
      [
      \tan 30^\circ=\frac{1}{\sqrt{3}}.
      ]
      所以直線 (CA) 的方程為
      [
      y=\frac{1}{\sqrt{3}}x. \tag{1}
      ]
    • 由 (\angle EDB=60^\circ) 且 (E) 在 (AD) 上（注意：(\angle EDB) 表示 (AD) 與 (DB) 的夾角），可得 (AD) 通過 (D(15,0))，且方向角為 (60^\circ)。因此直線 (AD) 的方程為
      [
      y=\sqrt{3},(x-15). \tag{2}
      ]
    • 聯立 (1) 與 (2) 求交點（即 (A)）：
      [
      \frac{1}{\sqrt{3}}x=\sqrt{3}(x-15)\ \Longrightarrow\ x=3(x-15)\ \Longrightarrow\ 2x=45,
      ]
      [
      \Rightarrow\ x=\frac{45}{2}=22.5.
      ]
      代回 (1) 得
      [
      y=\frac{1}{\sqrt{3}}\cdot\frac{45}{2}=\frac{15\sqrt{3}}{2}.
      ]
      因此
      [
      A=\Big(\tfrac{45}{2},\ \tfrac{15\sqrt{3}}{2}\Big).
      ]

    （圖 1：顯示 (C) 為原點，從 (C) 作 (30^\circ) 射線、從 (D) 作 (60^\circ) 射線，二者交點為 (A)。）

    補充計算（檢查長度）：
    [
    AC=\sqrt{\Big(\tfrac{45}{2}\Big)^2+\Big(\tfrac{15\sqrt3}{2}\Big)^2}
    =\tfrac{1}{2}\sqrt{45^2+15^2\cdot3}
    =\tfrac{1}{2}\sqrt{2700}=\tfrac{30\sqrt3}{2}=15\sqrt3.
    ]

    步驟 2：利用 (AD) 的方向求 (E)（圖 2）

    已知 (ED=7)，且 (E) 在 (AD) 上，且 (AD) 的方向是 (60^\circ)。沿 AD 的方向從 (D) 移動長度 7 即為 (E)：

    • 計算 (AD)：
      [
      AD=\sqrt{\Big(\tfrac{45}{2}-15\Big)^2+\Big(\tfrac{15\sqrt3}{2}\Big)^2}
      =\tfrac{15}{2}\sqrt{1+3}=15.
      ]
    • 因為 (AD=15)，且 (ED=7)，所以 (AE=AD-ED=8)。
    • 單位方向向量為 ((\cos60^\circ,\sin60^\circ)=(\tfrac12,\tfrac{\sqrt3}{2}))。因此
      [
      E=D+7\Big(\tfrac12,\tfrac{\sqrt3}{2}\Big)=\Big(15+\tfrac{7}{2},\ \tfrac{7\sqrt3}{2}\Big)=\Big(\tfrac{37}{2},\ \tfrac{7\sqrt3}{2}\Big).
      ]
    • 檢查：
      [
      \overrightarrow{EA}=A-E=\Big(\tfrac{45}{2}-\tfrac{37}{2},\ \tfrac{15\sqrt3}{2}-\tfrac{7\sqrt3}{2}\Big)=(4,\ 4\sqrt3),
      ]
      [
      |\overrightarrow{EA}|=\sqrt{4^2+(4\sqrt3)^2}=\sqrt{64}=8.
      ]

    （圖 2：顯示 (A,D,E) 在同一直線上，標明 (AD=15,\ ED=7,\ AE=8)。）

    步驟 3：以 (\angle AEB=120^\circ) 求 (B)（圖 3）

    利用向量內積把角度條件轉為代數方程：

    • 設 (B=(b,0))。則
      [
      \overrightarrow{EA}=(4,\ 4\sqrt3),\qquad
      \overrightarrow{EB}=\Big(b-\tfrac{37}{2},\ -\tfrac{7\sqrt3}{2}\Big).
      ]
    • 向量內積：
      [
      \overrightarrow{EA}\cdot\overrightarrow{EB}
      =4\cdot\frac{2b-37}{2}+(4\sqrt3)\cdot\Big(-\frac{7\sqrt3}{2}\Big)
      =4b-116.
      ]
    • 長度：
      [
      |\overrightarrow{EA}|=8,\qquad
      |\overrightarrow{EB}|=\tfrac12\sqrt{(2b-37)^2+147}.
      ]
    • 內積等式（(\cos120^\circ=-\tfrac12)）：
      [
      4b-116=8\cdot\frac12\sqrt{(2b-37)^2+147}\cdot\Big(-\tfrac12\Big).
      ]
      化簡得
      [
      116-4b=2\sqrt{(2b-37)^2+147}.
      ]
      平方並整理後得到
      [
      -336b+7392=0 \quad\Longrightarrow\quad b=22.
      ]
    • 因此 (B=(22,0))，且
      [
      BC=22,\qquad BD=22-15=7.
      ]
      （注意到 (BD=ED=7)，一個漂亮的對稱結果。）

    （圖 3：畫出 (E) 與 (B(22,0)) 的連線，並示 (\angle AEB=120^\circ)。）

    步驟 4：計算 (AB)（圖 4）

    • 現在 (A=\big(\tfrac{45}{2},\tfrac{15\sqrt3}{2}\big)) 與 (B=(22,0))。計算：
      [
      AB^2=\Big(\tfrac{45}{2}-22\Big)^2+\Big(\tfrac{15\sqrt3}{2}\Big)^2
      =\Big(\tfrac{1}{2}\Big)^2+\Big(\tfrac{15\sqrt3}{2}\Big)^2
      =\tfrac{676}{4}=169.
      ]
      [
      \Rightarrow AB=\sqrt{169}=13.
      ]

    （圖 4：完整三角形 (ABC)，標示 (AB=13)。）

    總結

    最終答案：(\boxed{AB=13})。
