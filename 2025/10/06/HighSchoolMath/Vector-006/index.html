三角函數直觀整理

發表於 2025-10-06 分類於 高中數學 閱讀次數：
用梯子、單位圓和小故事，把 sin/cos/tan 和其他三角函數整理，附數值範例與記憶口訣。

基本情境：想像一把梯子靠牆

把一把梯子靠在牆上，梯子的底端在地面上、頂端靠牆。把形成的直角三角形想清楚：

  • 斜邊（hypotenuse）：梯子的整段長度。
  • 對邊（opposite）：跟我們關心的角（比如梯子與地面的角）正對著的那一邊（就是牆上高度）。
  • 鄰邊（adjacent）：跟那個角相鄰、在地面上的那一段（就是我們往前走的距離）。

直觀定義（用梯子說話）

$$
設我們看一個角 (\theta)（就是梯子和地面成的角），那麼：

  • (\displaystyle \sin\theta=\frac{\text{對邊}}{\text{斜邊}})
    → 「sin」告訴你：梯子頂端離地面有多高，佔整把梯子長度的幾分之幾。
    試想：梯子越直（角越大），頂端越高，sin 越大。

  • (\displaystyle \cos\theta=\frac{\text{鄰邊}}{\text{斜邊}})
    → 「cos」告訴你：你從底端往前走的長度，佔整把梯子長度的幾分之幾。
    試想：角越小（梯子趴得平），cos 越接近 1。

  • (\displaystyle \tan\theta=\frac{\text{對邊}}{\text{鄰邊}}=\frac{\sin\theta}{\cos\theta})
    → 「tan」告訴你：每向前走 1 步（run），會上升多少步（rise）。（這就是我們說的 slope／斜率）

記憶口訣：SOH CAH TOA

  • SOH: ( \sin = \dfrac{Opp}{Hyp})
  • CAH: ( \cos = \dfrac{Adj}{Hyp})
  • TOA: ( \tan = \dfrac{Opp}{Adj})

三角函數的其他成員（簡單說）

  • (\displaystyle \csc\theta=\frac{1}{\sin\theta}=\frac{\text{斜邊}}{\text{對邊}})（csc = cosecant）
    → 如果 ( \sin ) 很小，( \csc ) 很大（因為是倒數）。
  • (\displaystyle \sec\theta=\frac{1}{\cos\theta}=\frac{\text{斜邊}}{\text{鄰邊}})（sec = secant）
  • (\displaystyle \cot\theta=\frac{1}{\tan\theta}=\frac{\text{鄰邊}}{\text{對邊}})（cot = cotangent）

這三個就是 sin, cos, tan 的「倒數朋友」。

單位圓的直觀（為什麼 sin/cos 會是座標）

想像一個半徑為 1 的圓（單位圓），在圓上選一個角 (\theta)（從正 x 軸逆時針量）。從圓心向外畫一條線到圓周，線與 x 軸夾角是 (\theta)。那條與圓交點的座標剛好是
[
(\cos\theta,\ \sin\theta).
]
直觀：沿著角度方向走到圓上，x（橫）是 cos，y（直）是 sin。因為半徑是 1，sin 與 cos 就成了「比例」或「座標」。
$$

幾個常見角度（數值範例，容易記）

$$
把下面記起來，常在考試出現：

角度 (\sin) (\cos) (\tan)
(30^\circ) (\tfrac{1}{2}) (\tfrac{\sqrt3}{2}) (\tfrac{1}{\sqrt3})（約 0.577）
(45^\circ) (\tfrac{\sqrt2}{2}) (\tfrac{\sqrt2}{2}) (1)
(60^\circ) (\tfrac{\sqrt3}{2}) (\tfrac{1}{2}) (\sqrt3)（約 1.732）

小故事幫記憶：

  • 30°：想像一把很斜、接近地面的梯子，頂端高度剛好是梯子長的一半（sin ＝ 1/2）。
  • 45°：斜上與前進一樣多（rise = run），所以 sin = cos，tan = 1。
  • 60°：比 30° 更直（高度較大），sin60 = √3/2。

重要關係（用簡單話說）

  • 畢氏恒等式（很重要、常用）：
    [
    \sin^2\theta+\cos^2\theta=1.
    ]
    這句話可以想成：「在單位圓上，x²+y²=1」。因為 ((x=\cos\theta), (y=\sin\theta))，所以自然成立。

  • tan 與 sin, cos 的關係：
    [
    \tan\theta=\frac{\sin\theta}{\cos\theta}.
    ]
    （也就是 slope = rise/run）。

  • 範圍：(\sin) 與 (\cos) 的值永遠在 ([-1,1]) 之內；(\tan) 可以是任何實數（但在 (\cos\theta=0) 時沒定義，例如 (90^\circ)）。

  • 符號（簡單）：

    • (\cos(-\theta)=\cos\theta)（cos 是「偶函數」，負角度不改變 cos）
    • (\sin(-\theta)=-\sin\theta)（sin 是「奇函數」，負角會把方向翻轉）

$$

$$

小練習（練習看懂）

  1. 一把長 10 的梯子，和地面夾角 30°，梯子頂端高度是多高？
    (\displaystyle \text{高度}=10\cdot\sin30^\circ=10\cdot\frac12=5)。

  2. 同一把梯子，底端向前走了多少？（run）
    (\displaystyle \text{run}=10\cdot\cos30^\circ=10\cdot\frac{\sqrt3}{2}=5\sqrt3)。

  3. 斜率（每向前 1 步上升多少）是多少？
    (\tan30^\circ=\dfrac{1}{\sqrt3})（約 0.577）。
    $$
