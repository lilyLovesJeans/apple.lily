金魚都能懂的網頁設計入門-CSS Reset

發表於 2024-06-13 更新於 2024-08-24 分類於 金魚都能懂的網頁設計入門-學習筆記 閱讀次數：
金魚都能懂的網頁設計入門-第十一天 -CSS Reset

CSS Reset : 觀看網頁的方式是瀏覽器有 firfox IE chrome , 解讀的方法及支持工能都不盡相同.

在每個瀏覽器 都有自建的 CSS 樣式去呈現網頁
,目的是萬一沒有自訂樣式會無法分辨。

網頁上的空白距離是 padding or margin 產生的?
還是設定在 html 的padding 或是設定在body 的margin？
這樣設定自己網頁的CSS會有點困擾.

CSS Reset 有點太over了一點 , 就是所有的元素都做的一模一樣。
去google 找CSS Reset可以找到

mayerWEB , 他所撰寫的 CSS Reset:

寫一html來測試 reset.CSS

還未裝飾之前，所有字體大小都變的一模一樣

用style重新裝飾網頁

Normalize.css 是什麼?

因為 reset.CSS把所有的預設都拿掉，導致後台編輯的困擾,所以只須因瀏覽器差異化，而做reset , 可以用Normalize.css.
但新手建議先用reset.css. 再慢慢去了解 Normalize.css

ps. 可以參考小艾的說明：小事之 CSS Reset 及 CSS normalize.
