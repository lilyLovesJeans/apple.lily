HTML 的標籤有

網頁會使用到的標籤有:
主標題，副標題，小標題，內文，無序清單 ，有序清單（步驟）， 導覽列， 超連結，粗體字，圖片(是單標籤<img>) ，<figcaption>有利 SEO 但須要隱藏（可用CSS ）, 表格,
粗體: <b>視覺上告知粗體字</b> <strong>強調-for SEO</strong> ,
斜體: <i>斜體字</i> <em>更大層度的強調emphasis-斜體 for SEO</em>
