2026-06-03 有資料庫支援的雲端免費平台 有那些?

我:
所以有資料庫支援的雲端免費平台 有那些?

GPT:
有多個雲端平台提供免費的資料庫服務，以下是一些提供免費資料庫選項的平台，可以用於學習、開發和測試小型應用：

1. Amazon Web Services (AWS)

  • AWS Free Tier 提供 12 個月的免費服務，包括資料庫選項：

    • Amazon RDS：免費使用一個微型實例（t2.micro），可用於 MySQL、PostgreSQL、MariaDB 等，每月 750 小時。
    • Amazon DynamoDB：每月有 25 GB 的存儲空間和 2000 萬次讀取和寫入請求，適合學習和使用非關係型資料庫。

AWS 的免費層適合初學者和開發者，能夠免費嘗試使用雲端資料庫。

2. Google Cloud Platform (GCP)

  • Google Cloud Platform 提供 12 個月的免費 $300 美元點數，可用於所有 GCP 服務。
  • Google Cloud SQL：可以使用免費點數來創建 MySQL、PostgreSQL、SQL Server 等資料庫。
  • Firebase Realtime Database：Firebase 提供了一些免費配額，適合小型應用的雲端資料庫需求。
  • BigQuery：每月提供 1 TB 的免費查詢配額。

GCP 提供的 $300 點數可以用於體驗多種類型的資料庫服務。

3. Microsoft Azure

  • Azure Free Account 提供 $200 美元點數，有效期 30 天，可以用於任何 Azure 服務。
  • Azure SQL Database：在免費層中有一定的配額，可用於試驗性的數據庫使用。
  • Azure Cosmos DB：提供 400 RU/s（請求單位每秒）和 5 GB 的永久免費層，適合開發和測試應用。

Azure 的免費帳戶可以幫助您快速部署和測試資料庫應用。

4. Oracle Cloud

  • Oracle Cloud Free Tier 提供永久免費的雲端服務，包括資料庫：

    • Oracle Autonomous Database：提供兩個 Autonomous Database 實例，每個 20 GB 的儲存空間，適合開發和測試使用。
    • Compute VM：也有永久免費的 VM，您可以自行部署 MySQL 等資料庫。

Oracle 的永久免費層是一個不錯的選擇，特別是對於需要使用 Oracle 資料庫或免費 VM 的開發者。

5. IBM Cloud

  • IBM Cloud Lite Plans 提供多種 Lite 版本的雲端服務，這些服務可以永久免費使用：

    • IBM Db2：提供 Lite 版本，包含 200 MB 的存儲空間。
    • Cloudant NoSQL Database：提供 1 GB 的免費存儲和一定數量的讀取和寫入請求。

IBM 的 Lite 計劃對於小型應用和測試環境特別合適，沒有時間限制。

6. Heroku

  • Heroku Postgres：Heroku 提供免費的 PostgreSQL 附加元件（Hobby Dev 計劃），適合測試和學習用途。
  • ClearDB MySQL 和 JawsDB MySQL：這些都是 Heroku 的第三方附加元件，可以提供 MySQL 資料庫的免費選擇，但資源有限。

Heroku 是 PaaS 平台，免費的資料庫選擇適合小型應用或測試環境。

7. DigitalOcean

  • DigitalOcean Managed Database：DigitalOcean 提供託管的資料庫服務，如 PostgreSQL、MySQL、Redis 等。
  • App Platform Free Plan：可以部署靜態網站或應用，並連接到外部的免費資料庫（例如外部的 Heroku Postgres）。

DigitalOcean 的資料庫不完全免費，但起價較低，並且提供良好的管理選項，適合個人開發者。

8. MongoDB Atlas

  • MongoDB Atlas 提供免費的託管 MongoDB 叢集，包含 512 MB 的存儲，適合用於測試和小型專案。
  • 這是一個 NoSQL 資料庫，特別適合需要靈活數據模型的應用。

9. Firebase by Google

  • Firebase Realtime Database 和 Firestore 都有免費層級，適合移動應用和 Web 應用。
  • 提供一定的資料存儲和請求配額，非常適合小型專案和學習。

Firebase 非常適合前端開發人員和需要快速後端搭建的應用。

總結

  • AWS、GCP、Azure：這些大型雲平台提供的免費層級可以讓您體驗雲端資料庫（如 MySQL、PostgreSQL 等）的所有功能，非常適合學習和測試。
  • Oracle Cloud：提供永久免費的資料庫服務，特別適合開發和測試。
  • Heroku：提供免費的 PostgreSQL，適合快速部署和測試小型 Web 應用。
  • IBM Cloud 和 MongoDB Atlas：提供永久免費的資料庫選擇，適合小型應用和個人學習。
  • Firebase：適合前端開發者，提供免費的實時資料庫和 Firestore，適合 Web 和行動應用。

這些免費的雲端資料庫選項能夠滿足大部分學習、開發和小型應用的需求。如果您的應用需要持續運行，或者有更高的性能需求，建議升級到付費計劃以獲得更多的資源和穩定性。
