問題回答

問題 1：if-else 語句和 switch 語句有什麼區別？什麼時候應該用 switch？

回答：

  • if-else 語句 適用於根據不同的條件進行判斷，通常適合於處理單一或複雜的邏輯表達式。例如，如果條件是布林值（true 或 false），或者條件是由多個比較和邏輯組合而成，if-else 更為合適。

    範例：

    bool loginOk = false;
if (loginOk)
{
    Console.WriteLine("登入成功!");
}
else
{
    Console.WriteLine("登入失敗!");
}

  • switch 語句 適用於多值選擇的情況，通常是當需要對一個變數進行多個可能值的判斷時使用，例如枚舉類型或一組特定的字串常量。這種語句在處理多個可能的選擇時比 if-else 更直觀和清晰。

    範例：

    string messageColor = "green";

switch (messageColor)
{
    case "blue":
        Console.WriteLine("藍色...一般訊息");
        break;
    case "green":
        Console.WriteLine("綠色....通過!");
        break;
    case "red":
        Console.WriteLine("紅色....失敗!!!");
        break;
    case "orange":
        Console.WriteLine("橘色....警告!!!");
        break;
    default:
        Console.WriteLine("未知的訊息!!!!");
        break;
}
    • 輸出：綠色….通過!

總結：

  • 當條件較複雜或涉及多個布林運算時，使用 if-else。
  • 當需要針對一個值的多種可能性進行選擇時，使用 switch 更合適。

問題 2：while 和 do-while 迴圈的主要區別是什麼？

回答：

  • while 迴圈：在執行迴圈內的代碼之前會先檢查條件是否為 true。只有當條件為 true 時，才會進行迴圈。如果條件一開始就是 false，那麼迴圈一次也不會執行。

    範例：

    int counter_1 = 1;
while (counter_1 <= 3)
{
    Console.WriteLine($"counter_1: {counter_1}");
    counter_1++;
}

    • 輸出：

      counter_1: 1
counter_1: 2
counter_1: 3

  • do-while 迴圈：首先執行迴圈內的代碼，然後再檢查條件是否為 true。無論條件最初是否成立，do-while 迴圈至少會執行一次。

    範例：

    int counter_2 = 1;
do
{
    Console.WriteLine($"counter_2: {counter_2}");
    counter_2++;
} while (counter_2 <= 3);

    • 輸出：

      counter_2: 1
counter_2: 2
counter_2: 3

總結：

  • while 迴圈在條件成立時進行，可能一次都不執行。
  • do-while 迴圈至少執行一次，然後根據條件繼續決定是否繼續。
