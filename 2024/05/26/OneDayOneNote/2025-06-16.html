2025-06-16 檢查 Mac B（MCP Server 機） 上的 Node.js 版本

很好 👍 我來一步一步帶你檢查 Mac B（MCP Server 機） 上的 Node.js 版本，確認你是否能直接用 fetch()。

Step 1 — 開啟終端機

在你的 Mac B 上，打開 Terminal（終端機）。

Step 2 — 檢查 Node.js 版本

在終端機輸入：

node -v

你會看到類似這樣的結果：

v20.11.1

Step 3 — 判斷

  • 如果結果是 v18.x.x 或 v20.x.x（或更新的版本） → ✅ 可以直接用 fetch()。

  • 如果結果是 v16.x.x 或更舊 → ❌ 沒有內建 fetch()，要額外安裝：

    npm install node-fetch

Step 4 — 驗證 fetch 是否可用（選做）

你也可以在終端機輸入：

node

進入互動模式，然後輸入：

await fetch("https://example.com").then(r => r.text());

如果能順利顯示網頁內容 → 表示 fetch() 可用。
