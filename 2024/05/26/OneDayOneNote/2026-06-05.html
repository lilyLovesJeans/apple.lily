在 macOS 上進入 MySQL，您可以使用終端機來進行操作。以下是如何使用命令行進入 MySQL 的步驟：

步驟 1：打開終端機

  1. 按下 Command + Space 鍵，打開 Spotlight 搜尋，然後輸入 「Terminal」，按下 Enter，這會打開終端機窗口。
  2. 您也可以從「應用程式」>「實用工具」中找到 「終端機」。

步驟 2：進入 MySQL

假設 MySQL 已經安裝並且服務正在運行，您可以使用以下命令進入 MySQL：

  1. 使用 root 使用者進入 MySQL：
    在終端機中輸入以下命令，並根據提示輸入密碼：

    mysql -u root -p
    • **-u root**：表示您以 root 使用者身份登錄。
    • **-p**：這將提示您輸入 MySQL root 使用者的密碼。

  2. 如果 MySQL 安裝在默認位置並且已經設置了 PATH 環境變數，這個命令應該能正常運行。輸入密碼之後，您就會進入 MySQL 提示符 **mysql>**。

  3. 連接到特定資料庫：
    如果您想進入特定的資料庫，可以在上面的命令後添加資料庫名稱：

    mysql -u root -p logisticsdb

    然後輸入您的密碼，就會直接進入該資料庫。

常見指令

  • 查看資料庫列表：

    SHOW DATABASES;

  • 選擇資料庫：

    USE database_name;

  • 查看資料表：

    SHOW TABLES;

  • 退出 MySQL：
    您可以輸入以下命令退出 MySQL 提示符：

    exit;

    或者輸入 **quit**。

啟動 MySQL 服務

如果您遇到 MySQL 無法連接的問題，可能是 MySQL 服務沒有啟動。可以使用以下命令來啟動 MySQL：

  1. 如果 MySQL 是使用 Homebrew 安裝的，啟動命令為：

    brew services start mysql

  2. 如果 MySQL 是使用 DMG 安裝包安裝的，您可以使用以下命令啟動：

    sudo /usr/local/mysql/support-files/mysql.server start

    輸入您的 macOS 使用者密碼來啟動 MySQL。

檢查 MySQL 安裝位置

如果您無法找到 mysql，可以檢查 MySQL 的安裝位置。可以使用以下命令來查找：

which mysql

如果返回的是類似 /usr/local/mysql/bin/mysql，則表示 MySQL 已安裝，並且您可以使用該路徑來進入 MySQL：

/usr/local/mysql/bin/mysql -u root -p

小結

  • 使用 mysql -u root -p 命令進入 MySQL。
  • 輸入密碼後可以進入 MySQL 的交互模式進行操作。
  • 如果 MySQL 服務未啟動，可以使用 brew services start mysql 或其他啟動命令。

這些步驟應該可以幫助您在 macOS 上順利進入 MySQL，如果有其他問題，歡迎隨時提問！
