“2026-05-25 想直接在 Windows 的命令提示字元（cmd）中使用 mysql＞ 來連接 MySQL 伺服器，您需要先安裝 MySQL”

GPT:
是的，如果您想直接在 Windows 的命令提示字元（cmd）中使用 mysql> 來連接 MySQL 伺服器，您需要先安裝 MySQL。通常，您需要安裝 MySQL Community Server，這包括 MySQL 伺服器以及用於命令行連接的 MySQL 客戶端工具（如 mysql.exe）。以下是具體的安裝和設置步驟：

1. 下載 MySQL 社群版

首先，您需要下載 MySQL Community Server。可以前往 MySQL 的官方下載頁面下載：

  • MySQL Community Server 下載頁面

選擇適合 Windows 的安裝程式，下載並安裝。

2. 安裝 MySQL

按照以下步驟安裝 MySQL：

  1. 執行下載的安裝程序，選擇安裝類型，建議選擇 “Server Only” 或 **”Custom”**，這樣可以確保您同時安裝了 MySQL 伺服器和客戶端工具。
  2. 在安裝過程中，您可以設置 MySQL root 用戶的密碼，並配置其他相關選項。

3. 添加 MySQL 到系統環境變數

為了能夠直接在 cmd 中使用 mysql 命令，您需要將 MySQL 的 bin 目錄添加到 Windows 的環境變數中。具體步驟如下：

  1. 打開 「開始」 菜單，並搜索 「環境變數」，然後點選 「編輯系統環境變數」。

  2. 點選 「環境變數」 按鈕。

  3. 在 「系統變數」 中找到 **Path**，並點選 「編輯」。

  4. 點選 「新增」，並輸入 MySQL 安裝目錄下的 bin 路徑，例如：

    C:\Program Files\MySQL\MySQL Server 8.0\bin

這樣做的目的是讓 Windows 系統知道 mysql.exe 的位置，使得您可以直接在命令提示字元中輸入 mysql 來啟動 MySQL 客戶端。

4. 使用命令行連接到 MySQL

設置完成後，您可以通過命令提示字元連接到 MySQL 伺服器。

  1. 打開 cmd（命令提示字元）。

  2. 輸入以下命令以連接到 MySQL 伺服器：

    mysql -u root -p

    這裡：

    • -u root 表示使用 root 使用者進行登錄。
    • -p 表示要求輸入密碼。

按下 Enter 後，會提示您輸入 MySQL root 使用者的密碼。輸入您在安裝過程中設置的密碼，即可進入 MySQL 互動模式，提示符會變成 **mysql>**。

5. 常見 MySQL 互動模式命令

  • 查看資料庫：

    SHOW DATABASES;

  • 使用某個資料庫：

    USE my_database;

  • 查看某個資料庫中的表：

    SHOW TABLES;

  • 執行查詢：

    SELECT * FROM my_table;

總結

如果您希望能夠在命令提示字元中使用 mysql> 命令，您需要安裝 MySQL Community Server，這樣就會包含 mysql 客戶端工具。安裝完成後，通過配置環境變數，您就可以在命令提示字元中直接輸入 mysql 來連接到 MySQL 資料庫了。
