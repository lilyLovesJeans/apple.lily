GPT：
在 C# 中，<T> 是泛型的類型參數聲明，它表示集合可以用來儲存任何類型的資料，例如基本資料類型（如 int、string）、自定義類型（如 Person 類別），甚至其他泛型類型的資料（如 List<List<int>>）。以下是一些具體例子：

1. 基本資料類型

List<int> intList = new List<int>();     // 儲存整數
List<string> stringList = new List<string>(); // 儲存字串

2. 自定義類型

假設有一個 Person 類別，可以使用泛型 List<T> 來儲存多個 Person 物件：

public class Person
{
    public string Name { get; set; }
    public int Age { get; set; }
}

List<Person> people = new List<Person>(); // 儲存 Person 類別物件

3. 其他泛型類型

泛型也可以嵌套使用，例如儲存其他泛型集合的資料：

List<List<int>> listOfLists = new List<List<int>>(); // 儲存 List<int> 的列表

4. 物件型別

泛型集合也能儲存任何繼承自 object 的類型，因為所有類別在 C# 中都繼承自 object：

List<object> objectList = new List<object>(); // 可以儲存不同類型的物件
objectList.Add(1);         // 加入整數
objectList.Add("Hello");    // 加入字串
objectList.Add(new Person { Name = "Alice", Age = 30 }); // 加入 Person 類別物件

使用泛型 T 提供了更高的靈活性和型別安全性，讓你可以使用單一泛型類別來處理多種不同的資料類型。
