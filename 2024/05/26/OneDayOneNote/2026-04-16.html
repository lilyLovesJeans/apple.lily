2026-04-16 MySQL JS > ? 要怎麼連?

我：
MySQL JS > ? 要怎麼連?

GPT：
MySQL Shell 的 JS 模式 是一種可以使用 JavaScript 語法來管理和操作 MySQL 伺服器的方式。MySQL Shell 支援多種模式，包括 SQL 模式 和 JS 模式，可以讓您選擇最熟悉的語言來與 MySQL 進行互動。
以下是如何在 MySQL Shell（JS 模式） 中連接 MySQL 資料庫的步驟：

  1. 啟動 MySQL Shell
    首先，您需要在終端機或命令提示字元（cmd）中打開 MySQL Shell。您可以通過在命令提示字元中輸入 mysqlsh 來啟動 MySQL Shell。
    mysqlsh
  2. 切換到 JS 模式
    MySQL Shell 默認可能是 SQL 模式（命令提示符為 MySQL>）。如果要切換到 JS 模式，可以輸入以下命令：
    \js
    或者直接在 MySQL Shell 啟動時使用 –js 標誌啟動：
    mysqlsh –js
  3. 使用 JavaScript 連接到 MySQL 資料庫
    在 JS 模式 下，您可以使用 JavaScript 語法來連接到 MySQL 資料庫。使用 dba.getCluster()、mysql.getSession() 等方法來進行連接和操作。
    以下是基本的連接方式：
    // 建立連接到 MySQL 資料庫的 session
    var session = mysql.getSession({
    user: ‘your_username’,
    password: ‘your_password’,
    host: ‘your_server_ip’,
    port: 3306 // 預設 MySQL 埠號
    });
    將以上代碼中的 your_username、your_password 和 your_server_ip 分別替換為您的 MySQL 用戶名、密碼和 MySQL 伺服器的 IP 地址。
    例如：
    var session = mysql.getSession({
    user: ‘root’,
    password: ‘mypassword’,
    host: ‘192.168.1.100’,
    port: 3306
    });
  4. 執行 SQL 查詢
    連接建立後，您可以使用 session.run() 來執行 SQL 查詢。例如：
    // 選擇要使用的資料庫
    session.sql(“USE my_database”).execute();

// 執行查詢
var result = session.sql(“SELECT * FROM my_table”).execute();
print(result);
這裡 session.sql() 方法用於執行 SQL 語句，並返回結果。
5. 關閉連接
完成資料庫操作後，您可以關閉 session：
session.close();
全部過程示範
完整的代碼如下：
// 連接到 MySQL 資料庫
var session = mysql.getSession({
user: ‘root’,
password: ‘mypassword’,
host: ‘192.168.1.100’,
port: 3306
});

// 選擇資料庫
session.sql(“USE my_database”).execute();

// 查詢資料
var result = session.sql(“SELECT * FROM my_table”).execute();
print(result);

// 關閉連接
session.close();
JS 模式的用途
1. 自動化管理：JS 模式可以用於自動化管理任務，尤其是在配置 MySQL InnoDB Cluster 或進行複製設置時。 
2. 腳本執行：因為 JS 模式支援 JavaScript 語法，可以將日常管理操作寫成腳本，並通過 MySQL Shell 運行，提升效率。 
3. 複製和集群管理：JS 模式對於管理 MySQL InnoDB Cluster 來說非常有用，提供了一些內建的 dba 物件來進行管理操作。 
這樣，您就可以在 MySQL Shell 的 JS 模式下連接並管理您的 MySQL 資料庫，利用 JavaScript 的靈活性進行各種操作。

我：
那我要直接下 cmd mysql > , 是要安裝 mysql ?

GPT:
是的，如果您想直接在 Windows 的命令提示字元（cmd）中使用 mysql> 來連接 MySQL 伺服器，您需要先安裝 MySQL。通常，您需要安裝 MySQL Community Server，這包括 MySQL 伺服器以及用於命令行連接的 MySQL 客戶端工具（如 mysql.exe）。以下是具體的安裝和設置步驟：

1. 下載 MySQL 社群版

首先，您需要下載 MySQL Community Server。可以前往 MySQL 的官方下載頁面下載：

  • MySQL Community Server 下載頁面

選擇適合 Windows 的安裝程式，下載並安裝。

2. 安裝 MySQL

按照以下步驟安裝 MySQL：

  1. 執行下載的安裝程序，選擇安裝類型，建議選擇 “Server Only” 或 **”Custom”**，這樣可以確保您同時安裝了 MySQL 伺服器和客戶端工具。
  2. 在安裝過程中，您可以設置 MySQL root 用戶的密碼，並配置其他相關選項。

3. 添加 MySQL 到系統環境變數

為了能夠直接在 cmd 中使用 mysql 命令，您需要將 MySQL 的 bin 目錄添加到 Windows 的環境變數中。具體步驟如下：

  1. 打開 「開始」 菜單，並搜索 「環境變數」，然後點選 「編輯系統環境變數」。

  2. 點選 「環境變數」 按鈕。

  3. 在 「系統變數」 中找到 **Path**，並點選 「編輯」。

  4. 點選 「新增」，並輸入 MySQL 安裝目錄下的 bin 路徑，例如：

    C:\Program Files\MySQL\MySQL Server 8.0\bin

這樣做的目的是讓 Windows 系統知道 mysql.exe 的位置，使得您可以直接在命令提示字元中輸入 mysql 來啟動 MySQL 客戶端。

4. 使用命令行連接到 MySQL

設置完成後，您可以通過命令提示字元連接到 MySQL 伺服器。

  1. 打開 cmd（命令提示字元）。

  2. 輸入以下命令以連接到 MySQL 伺服器：

    mysql -u root -p

    這裡：

    • -u root 表示使用 root 使用者進行登錄。
    • -p 表示要求輸入密碼。

按下 Enter 後，會提示您輸入 MySQL root 使用者的密碼。輸入您在安裝過程中設置的密碼，即可進入 MySQL 互動模式，提示符會變成 **mysql>**。

5. 常見 MySQL 互動模式命令

  • 查看資料庫：

    SHOW DATABASES;

  • 使用某個資料庫：

    USE my_database;

  • 查看某個資料庫中的表：

    SHOW TABLES;

  • 執行查詢：

    SELECT * FROM my_table;

總結

如果您希望能夠在命令提示字元中使用 mysql> 命令，您需要安裝 MySQL Community Server，這樣就會包含 mysql 客戶端工具。安裝完成後，通過配置環境變數，您就可以在命令提示字元中直接輸入 mysql 來連接到 MySQL 資料庫了。
