2025-04-27-Volar 0.0.15

Volar 0.0.15 Plan, Review & Build with AI , 有什麼新功能？

簡短答覆 — 我沒找到針對 0.0.15 的獨立「變更日誌 (changelog)」，但 0.0.15 在 Visual Studio Marketplace / VSIX 目錄上的擴充說明（也就是該版本在市集中登錄時的說明）顯示這個版本就是把 Volar — “Plan, Review & Build with AI” 作為一個 AI 協作／任務編排器推出，主要功能如下（來源：Marketplace 說明與 VSIX 列表）。(marketplace.visualstudio.com)

主要新功能（或重點功能，一般會在 0.0.15 中可用）

  • 任務視圖 (Task View)：在 VS Code 側邊欄新增「任務」視圖，可以建立/檢視/編輯任務（標題、內容、子任務等）。(marketplace.visualstudio.com)

  • 三步式 AI 工作流程：Plan → Breakdown → Execute

    • Plan：讓你給 AI 一個任務標題或簡述，AI 幫你產出實作計畫。
    • Breakdown：把大型/模糊任務切成小子任務。
    • Execute：在你審核同意計畫後，指示 AI 實際產出程式碼或命令。(marketplace.visualstudio.com)

  • MCP（Model Context Protocol）整合：支援透過 MCP 與多個 AI 工具串接（Marketplace 說明特別列出 Cursor、Windsurf、Claude Code，其他支援 MCP 的工具也可設定）。根據所用工具，Volar 會把 prompt 複製到剪貼簿或透過 terminal 傳送命令給某些工具。(marketplace.visualstudio.com)

  • 平台/環境提醒與已知問題：官方說明標註 以 macOS 為主要測試環境（其他平台可能遇到行為差異），以及常見的 MCP 設定/連線 troubleshooting。(marketplace.visualstudio.com)

  • VSIX / 市集上線資訊：0.0.15 的 VSIX 檔案已在第三方 VSIX 索引站點登錄（可離線下載安裝）。(VsixHub.com)
