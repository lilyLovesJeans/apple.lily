2025-08-09 Line 訂單系統- 升級到 Homebrew Git*

  • 版本：2.24.3 (Apple Git-128)
  • 路徑：應該是 /usr/bin/git（macOS 內建）

這台 Mac 也建議 升級到 Homebrew Git，跟你第一台保持一致，這樣開發環境才統一，避免版本差異導致的問題。

步驟（跟你第一台 Mac 一樣）

  1. 先確認是否已安裝 Homebrew
brew --version
  • 若顯示版本號，代表已安裝
  • 若沒有，先安裝 Homebrew：
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  1. 安裝最新版 Git
brew install git

（若已安裝舊版 Homebrew Git，可改成 brew upgrade git）

  1. 確認你的 Mac 架構（Intel 或 Apple Silicon）
uname -m
  • x86_64 → Intel → Homebrew Git 在 /usr/local/bin/git
  • arm64 → Apple Silicon → Homebrew Git 在 /opt/homebrew/bin/git
  1. 修改 PATH 讓系統優先使用 Homebrew Git
  • Intel Mac：
echo 'export PATH="/usr/local/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
  • Apple Silicon：
echo 'export PATH="/opt/homebrew/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
  1. 確認切換成功
which git
git --version

你應該會看到 /usr/local/bin/git 或 /opt/homebrew/bin/git，版本更新到 Homebrew 安裝的 Git（例如 2.52+ 或最新）。
