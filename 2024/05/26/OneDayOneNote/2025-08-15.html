2025-08-15 Line 訂單系統- 怎麼看 brew 的版本?

要查看 Homebrew（brew）本身的版本 很簡單，你只需要在終端機輸入：

✅ 查看 brew 版本

brew --version

範例輸出可能像：

Homebrew 4.3.3
Homebrew/homebrew-core (git revision xxxxx)
Homebrew/homebrew-cask (git revision xxxxx)

✅ 查看 brew 是否正常運作（增強版）

brew doctor

如果出現：

Your system is ready to brew.

代表沒問題。

⛽ 額外常用跟 brew 有關的版本指令

✔ 查看某個套件的版本（例如 git）

brew info git

✔ 查看某個已安裝套件的版本（例如 node）

brew list --versions node

✔ 查看所有已安裝套件 + 版本

brew list --versions

如果你告訴我你的輸出，我可以判斷：

  • 你的 Homebrew 是否太舊
  • 是否要更新
  • 你的 PATH 是否正確配置（特別是在 Intel vs M1/M2 Mac 很容易錯）
