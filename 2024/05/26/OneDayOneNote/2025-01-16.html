如何 run Vite 專案

🏃 如何 run Vite 專案？

  1. 開啟你的專案資料夾（例如在 VS Code）

  2. 在終端機輸入

npm install

👉（只需要做一次，安裝 package.json 裡的套件）

  1. 啟動開發伺服器
npm run dev
  1. 啟動後，終端機會出現網址
    通常是：
➜  Local:   http://localhost:5173/

直接點網址，或複製到瀏覽器打開就可以看了！

❓如果 npm run dev 失敗了？

可能是你的電腦「還沒安裝 Vite 專案」或「沒裝 node.js」

  • 確定安裝 Node.js（可以打 node -v 看有沒有版本）
  • 如果是全新開的空資料夾，要先建 vite 專案：
npm create vite@latest

（它會引導你選 Vue / React / 等等）
