2025-08-15 Line 訂單系統- 怎麼看 brew 的版本?

要查看你的 Mac 是否 已經安裝 nvm，有三種最簡單的方法。

✅ 方法 1：直接輸入指令

在 Terminal 輸入：

nvm --version

如果看到版本號，例如：

0.39.7

➡ 代表 nvm 已安裝

如果出現：

command not found: nvm

➡ 代表 沒有安裝 或 安裝後尚未設定好 ~/.zshrc 或 ~/.bashrc

✅ 方法 2：查看你的 Shell 設定檔

輸入：

cat ~/.zshrc

或（如果你用 bash）

cat ~/.bashrc

找到下面這段內容就表示已安裝：

export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"

如果沒有看到，就表示 nvm 還沒正確設定。

✅ 方法 3：查看 nvm 目錄是否存在

ls ~/.nvm

如果看到一大堆檔案（例如 nvm.sh、versions/），代表已安裝。

如果看到：

ls: cannot access '/Users/你/.nvm': No such file or directory

→ 代表未安裝。
