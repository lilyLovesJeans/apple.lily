2024-08-10 如何更改kenwheeler/slick 的箭頭顏色

可以在偽元素裹更改顏色, 記得要加上 !important 字樣，才可以覆寫原來的CSS

.slick-prev:before {
  color: red !important;
  background-color: #eee;
}
.slick-next:before {
  color: red !important;
  background-color: #eee;
}
