階段 1：Hello World 程式
解釋
在 C# 中，撰寫「Hello World」程式的目的是讓你了解最基本的 C# 程式結構，以及如何顯示訊息到螢幕上。在這個階段，我們會介紹 C# 程式的組成部分。
Hello World 範例
using System;

class Program
{
static void Main(string[] args)
{
Console.WriteLine(“Hello, World!”);
}
}
程式結構解析
1. using System;
◦ 這行是命名空間的引用。System 是 C# 中的基礎命名空間，其中包含許多常用的類別和方法，例如 Console。 
2. class Program
◦ Program 是類別的名稱。在 C# 中，所有程式碼都需要包含在一個類別裡。 
3. static void Main(string[] args)
◦ Main 方法是程式的入口點，每當程式開始執行時，會先從這個方法開始。 
◦ static：表示這個方法是靜態的，可以在沒有創建物件的情況下直接呼叫。 
◦ void：表示這個方法不會返回任何值。 
◦ string[] args：這是方法的參數，用於接收命令列參數（不必擔心這個，稍後會詳細介紹）。 
4. Console.WriteLine(“Hello, World!”);
◦ Console 是用於與控制台進行互動的類別。 
◦ WriteLine 方法將文本輸出到控制台，並在最後自動換行。 
◦ “Hello, World!” 是要顯示的訊息。 
執行方式
你可以使用 Visual Studio 或 Visual Studio Code 等開發工具來編寫和執行這段程式碼。當執行時，控制台會顯示：
Hello, World!
