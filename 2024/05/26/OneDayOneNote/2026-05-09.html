2026-05-09 C# 有 if elseif 嗎?

我:
C# 有 if elseif 嗎?

GPT:
是的，C# 支持 if-else if 結構，這樣可以用來處理多個條件，而不是僅限於兩個條件。if-else if 結構非常有用，當你有多個不同的條件需要依次檢查時，可以使用這種方式來進行更靈活的判斷。

if-else if 語句的語法

if-else if 的語法如下：

if (條件1)
{
    // 當條件1為true時執行的代碼
}
else if (條件2)
{
    // 當條件1為false且條件2為true時執行的代碼
}
else if (條件3)
{
    // 當條件1和條件2都為false且條件3為true時執行的代碼
}
else
{
    // 當所有條件都為false時執行的代碼
}

if-else if 的範例

假設你想要根據考試成績給出不同的評價，可以用 if-else if 語句來處理：

int score = 85;

if (score >= 90)
{
    Console.WriteLine("優秀");
}
else if (score >= 75)
{
    Console.WriteLine("良好");
}
else if (score >= 60)
{
    Console.WriteLine("及格");
}
else
{
    Console.WriteLine("不及格");
}

在這個範例中：

  • 如果 score 大於等於 90，將輸出「優秀」。
  • 如果 score 在 75 到 89 之間，將輸出「良好」。
  • 如果 score 在 60 到 74 之間，將輸出「及格」。
  • 如果 score 小於 60，將輸出「不及格」。

使用 if-else if 的情境

當有多個條件需要依次檢查且每個條件都彼此不同時，if-else if 是非常合適的。例如：

  • 根據分數等級給出評價。
  • 根據使用者輸入的數值選擇不同的行為。
  • 檢查多個條件，找出符合的特定情況。

if-else if 語句比單純的 if-else 語句更靈活，因為你可以用來處理更多的情況。它是 switch 語句的另一種選擇，適合處理一些複雜的邏輯判斷（特別是條件不是單一值比較的情況）。
