1. 方法的定義與結構

方法 是一段可以被重複執行的代碼塊，通常用於執行特定的操作。

  • 方法的結構：

    修飾詞 返回值類型 方法名(參數列表)
{
    // 方法內部的代碼
}
    • 修飾詞：定義方法的訪問級別，例如 public（公開）、private（私有）等。
    • 返回值類型：方法返回的數據類型。如果方法不返回任何數據，使用 void。
    • 方法名：方法的名稱，通常使用駝峰式命名法。
    • 參數列表：用於傳遞給方法的變數，可以有多個參數，如果沒有參數則留空。

2. 方法的參數與返回值

  • 參數 用於向方法傳遞數據，方法可以根據這些參數來執行操作。

  • 返回值 是從方法返回的結果。使用 return 關鍵字來返回結果。

  • 範例：

    public int Add(int a, int b)
{
    return a + b;
}

    在這個範例中：

    • public 是訪問修飾詞，表示這個方法可以在其他類中訪問。
    • int 是返回值類型，表示這個方法會返回一個整數。
    • Add 是方法名。
    • (int a, int b) 是參數列表，表示這個方法需要兩個整數作為參數。

3. 方法的調用

當方法定義好之後，我們就可以在程式中通過調用它來使用它。調用方法時需要傳入與參數列表匹配的參數。

  • 範例：

    class Program
{
    static void Main(string[] args)
    {
        int result = Add(3, 5);
        Console.WriteLine($"結果是: {result}");
    }

    public static int Add(int a, int b)
    {
        return a + b;
    }
}

  • 在 Main 方法中，我們調用了 Add(3, 5)，並將返回的結果存入變數 result，然後輸出結果。

4. 方法的範例

讓我們來看看一些不同類型的方法：

  1. 沒有返回值且沒有參數的方法（void 方法）：

    public void Greet()
{
    Console.WriteLine("Hello, welcome!");
}

    調用範例：

    Greet();

  2. 有返回值的方法：

    public double Multiply(double x, double y)
{
    return x * y;
}

    調用範例：

    double product = Multiply(4.5, 2.0);
Console.WriteLine($"乘積是: {product}");

  3. 有參數但沒有返回值的方法：

    public void PrintMessage(string message)
{
    Console.WriteLine($"Message: {message}");
}

    調用範例：

    PrintMessage("這是一個測試訊息");

  4. 有多個參數的方法：

    public int Subtract(int a, int b)
{
    return a - b;
}

    調用範例：

    int difference = Subtract(10, 4);
Console.WriteLine($"差值是: {difference}");
