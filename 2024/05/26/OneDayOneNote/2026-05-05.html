1. 什麼是變數？

變數 是一個用來存儲數據的命名空間。在程式中，變數可以幫助我們存放和管理各種數據，例如整數、浮點數、文字等等。你可以把變數想像成程式中的一個「小盒子」，可以在其中存放資料，並在程式的不同部分中使用這些資料。

範例：

int age = 25;
string name = "Alice";
double height = 5.9;

在上面的程式碼中，我們定義了三個變數：

  • **age**：用來存放一個整數（int）值 25。
  • **name**：用來存放一個字串（string）值 "Alice"。
  • **height**：用來存放一個雙精度浮點數（double）值 5.9。

2. 常見的資料型別

C# 提供了多種資料型別，用來存放不同類型的資料。在這裡，我們介紹一些最常見的基本資料型別：

  • 整數型別（int）：用來存放整數值。

    int number = 10;

  • 浮點數型別（float、double）：用來存放小數。float 的精度較低，而 double 的精度較高。

    double price = 19.99;
float weight = 3.5f; // 注意：`float` 型別的值需要加上 `f` 字尾

  • 布林型別（bool）：用來存放布林值（true 或 false）。

    bool isHappy = true;

  • 字串型別（string）：用來存放文字。

    string greeting = "Hello, World!";

  • 字符型別（char）：用來存放單個字符，必須使用單引號。

    char letter = 'A';

3. 變數的命名規則

在 C# 中，變數的命名需要遵循一些基本的規則：

  1. 名稱必須以字母或下劃線（_）開頭，不能以數字開頭。
  2. 變數名稱不能包含空格，例如 my variable 是不合法的名稱。
  3. 變數名稱區分大小寫，例如 age 和 Age 是兩個不同的變數。
  4. 避免使用 C# 關鍵字（如 int、class 等）作為變數名稱。

良好的變數命名風格：

  • 使用有意義的名稱，描述變數存放的內容。
  • 遵循駝峰命名法，例如 firstName 或 totalPrice。

範例：

int totalScore = 100;
string userName = "Bob";
bool isLoggedIn = false;
