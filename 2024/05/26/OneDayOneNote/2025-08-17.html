‘2025-08-17 Line 訂單系統- bash: nvm: command not found 系統找不到 nvm’

很好！你目前使用的是：

✅ Bash（/bin/bash）

所以你要設定的檔案不是 ~/.zshrc，而是：

✔ ~/.bash_profile

你的 nvm 已經安裝成功，只是「沒有載入到 bash」，所以系統找不到 nvm。

我們現在把設定補上即可。

📌 請完全照著我下面這 3 步走，不會出錯

Step 1：開啟 ~/.bash_profile

在 Terminal 輸入：

nano ~/.bash_profile

這會打開編輯器（黑色畫面）。

Step 2：把下面三行貼進去（Intel Mac 專用！）

請複製並貼到檔案最下面：

export NVM_DIR="$HOME/.nvm"
[ -s "/usr/local/opt/nvm/nvm.sh" ] && \. "/usr/local/opt/nvm/nvm.sh"
[ -s "/usr/local/opt/nvm/etc/bash_completion.d/nvm" ] && \. "/usr/local/opt/nvm/etc/bash_completion.d/nvm"

⚠ 路徑一定要 /usr/local/opt/nvm
（你是 Intel，不可用 /opt/homebrew）

Step 3：存檔 + 套用設定

在 nano 裡：

  1. 按 Ctrl + O（字母 O）→ 儲存
  2. 按 Enter
  3. 按 Ctrl + X → 離開編輯器

然後在 Terminal 輸入：

source ~/.bash_profile

✔ 完成後測試

nvm --version
nvm ls

你預期會看到：

0.40.3
->      v18.x.x

📣 貼給我：

nvm --version
nvm ls

我確認你的 nvm 是否成功啟動並載入。
