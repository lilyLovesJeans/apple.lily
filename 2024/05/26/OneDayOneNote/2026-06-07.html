我:
sudo /usr/local/mysql/bin/mysql -u root -p

GPT:
請輸入這條命令後，系統應該會提示您輸入 MySQL root 使用者的密碼：

sudo /usr/local/mysql/bin/mysql -u root -p

當看到提示 Enter password: 時，請輸入 root 使用者的密碼。如果您不確定這個密碼是什麼或者已經忘記密碼，您可以按照以下步驟重置 root 密碼。

重置 MySQL Root 密碼步驟

  1. 停止 MySQL 服務：
    首先停止 MySQL 服務，以便能夠在不檢查權限的模式下啟動 MySQL。

    sudo /usr/local/mysql/support-files/mysql.server stop

  2. 安全模式啟動 MySQL：
    使用安全模式啟動 MySQL，跳過權限檢查，允許您無需密碼進行登錄。

    sudo /usr/local/mysql/bin/mysqld_safe --skip-grant-tables &

  3. 無密碼進入 MySQL：
    在安全模式下，使用以下命令無需密碼進入 MySQL：

    sudo /usr/local/mysql/bin/mysql -u root

  4. 重置 Root 密碼：
    進入 MySQL 提示符後，執行以下命令來重置 root 密碼：

    USE mysql;
UPDATE user SET authentication_string = PASSWORD('new_password') WHERE User = 'root';
FLUSH PRIVILEGES;

    請將 new_password 替換為您希望設置的 root 密碼。

  5. 重新啟動 MySQL 服務：
    退出 MySQL 提示符後，重新啟動 MySQL 服務：

    sudo /usr/local/mysql/support-files/mysql.server start

  6. 使用新密碼登錄 MySQL：
    服務啟動後，嘗試使用新密碼登錄 MySQL：

    sudo /usr/local/mysql/bin/mysql -u root -p

    然後輸入剛剛設置的新密碼。

小結

  • 輸入 sudo /usr/local/mysql/bin/mysql -u root -p 後，輸入 root 密碼登錄 MySQL。
  • 如果忘記 root 密碼，可以按照上述步驟重置。
  • 重置過程需要停止 MySQL 服務，進入安全模式，然後重置密碼。
