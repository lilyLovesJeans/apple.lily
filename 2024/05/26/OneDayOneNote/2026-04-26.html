很好，你提到的這個現象是 C# 在最近的版本（C# 9 及以上）中引入了一個叫做 「頂層語句（Top-level Statements）」 的新特性，這使得撰寫和執行簡單的 C# 程式變得更加容易，尤其適合像「Hello World」這樣的入門範例。
頂層語句（Top-level Statements）
在 C# 9 及以上的版本中，你可以不需要手動撰寫 using、class Program、static void Main(string[] args) 等標準結構，而是直接撰寫程式的主要內容。編譯器會自動為你生成這些必要的結構。
所以你只需要這樣寫：
Console.WriteLine(“Hello, World!”);
這樣的程式同樣是有效的，因為 C# 編譯器會自動幫你補充必要的部分。這對於學習和快速編寫簡單程式非常方便。
背後的結構
雖然你只需要撰寫 Console.WriteLine(“Hello, World!”);，但實際上編譯器會幫你自動生成一個類別和 Main 方法，就像是這樣：
using System;

class Program
{
static void Main(string[] args)
{
Console.WriteLine(“Hello, World!”);
}
}
這樣做的好處是可以讓初學者更容易上手，而不用擔心額外的語法細節，讓你專注於學習 C# 的核心概念。
