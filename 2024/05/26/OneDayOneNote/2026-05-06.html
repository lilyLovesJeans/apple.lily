1. 算術運算符

算術運算符 用於進行基本的數學運算，如加、減、乘、除等。以下是常見的算術運算符：

  • 加法（+）：

    int a = 5;
int b = 3;
int sum = a + b; // 8

  • 減法（-）：

    int difference = a - b; // 2

  • 乘法（*）：

    int product = a * b; // 15

  • 除法（/）：除法運算符將兩數相除。

    int quotient = a / b; // 1 (整數除法，結果為整數)

    注意：如果需要小數，可以使用 double 或 float。

  • 取餘數（%）：用於求兩數相除後的餘數。

    int remainder = a % b; // 2

2. 比較運算符

比較運算符 用於比較兩個值，並返回布林值（true 或 false）。常見的比較運算符有：

  • 等於（==）：

    bool isEqual = (a == b); // false

  • 不等於（!=）：

    bool isNotEqual = (a != b); // true

  • 大於（>）：

    bool isGreater = (a > b); // true

  • 小於（<）：

    bool isLess = (a < b); // false

  • 大於等於（>=）：

    bool isGreaterOrEqual = (a >= b); // true

  • 小於等於（<=）：

    bool isLessOrEqual = (a <= b); // false

3. 邏輯運算符

邏輯運算符 用於組合布林值和進行邏輯操作，常見的邏輯運算符有：

  • 與（&&）：只有當兩個條件都為 true 時，結果才為 true。

    bool isAdult = true;
bool hasID = false;
bool canEnter = isAdult && hasID; // false

  • 或（||）：只要其中一個條件為 true，結果就為 true。

    bool hasTicket = true;
bool isVIP = false;
bool canAttend = hasTicket || isVIP; // true

  • 非（!）：取反，將 true 變成 false，或將 false 變成 true。

    bool isLocked = false;
bool isOpen = !isLocked; // true

4. 賦值運算符

賦值運算符 用於將某個值賦予變數，最常見的是等號 =，此外還有一些其他賦值運算符，可以對變數進行賦值和運算的組合操作：

  • 等號（=）：

    int x = 10;

  • 加等於（+=）：將變數增加某個值。

    x += 5; // x 現在為 15

  • 減等於（-=）：將變數減去某個值。

    x -= 3; // x 現在為 12

  • 乘等於（*=）：將變數乘以某個值。

    x *= 2; // x 現在為 24

  • 除等於（/=）：將變數除以某個值。

    x /= 4; // x 現在為 6

5. 運算符的使用範例

以下是一個包含算術運算、比較運算和邏輯運算的綜合範例：

int score = 85;
int passMark = 50;
bool hasPassed = score >= passMark; // true

int bonus = 10;
score += bonus; // score 現在為 95

bool hasCertificate = true;
bool canGetJob = hasPassed && hasCertificate; // true
Console.WriteLine($"Can the person get the job? {canGetJob}");

在這個範例中，我們使用了多種運算符來進行加分、比較成績以及決定是否能獲得工作。
