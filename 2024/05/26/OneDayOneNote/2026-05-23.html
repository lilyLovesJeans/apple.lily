1. 什麼是泛型？

泛型（Generics）允許我們在定義類別、方法或接口時，使用一個或多個未確定類型的參數。這樣可以在不具體指定數據類型的情況下創建靈活且可重用的代碼。

  • 性能優化：泛型可以避免裝箱和拆箱操作（例如在使用集合時），從而提高性能。

1. 什麼是泛型？

  • 代碼重用：同一泛型類別或方法可以處理多種數據類型，減少重複代碼。
  • 性能優化：使用泛型可以避免裝箱和拆箱操作，從而提高性能。

3. 常見的泛型集合類別

  • **List**：一種動態數組，可以存儲多個相同類型的對象。

    定義一個泛型類 Storage<T>，並在裡面添加一個方法 AddItem(T item) 用來添加項目，以及一個方法 GetItems() 返回所有項目。

    public class Storage<T>
{
    private List<T> items = new List<T>();

    public void AddItem(T item)
    {
        items.Add(item);
    }

    public List<T> GetItems()
    {
        return items;
    }
}

  2. 使用泛型方法來執行基本的數據操作

    定義一個泛型方法 FindMax<T>(List<T> items)，返回列表中的最大值。這要求這些項目必須實作 IComparable 接口。

    public T FindMax<T>(List<T> items) where T : IComparable
{
    if (items == null || items.Count == 0)
        throw new ArgumentException("The list cannot be null or empty.");

    T max = items[0];

    foreach (T item in items)
    {
        if (item.CompareTo(max) > 0)
            max = item;
    }

    return max;
}
