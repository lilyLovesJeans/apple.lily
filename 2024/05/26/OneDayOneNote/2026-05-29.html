2026-05-29 要連接到 MySQL 伺服器

我:
connect lily_Admin@192.168.0.104:3306
GPT:
要連接到 MySQL 伺服器，請在 MySQL Shell 中使用以下命令來連接到遠程伺服器上的資料庫：

\connect lily_Admin@192.168.0.104:3306

輸入這條指令後，MySQL Shell 會提示您輸入密碼。輸入您之前設定的 lily_Admin 使用者的密碼即可完成連接。

具體步驟：

  1. 打開 MySQL Shell：

    mysqlsh

  2. 使用 \connect 指令連接到伺服器：

    \connect lily_Admin@192.168.0.104:3306

  3. 輸入密碼（根據您之前的設置，應該是 1234）：

    Please provide the password for 'lily_Admin@192.168.0.104': ********

成功連接後，您就可以在 MySQL> 提示符下執行 SQL 語句，或者切換到其他模式（如 JavaScript 模式）進行操作。

如果連接失敗，請檢查以下幾點：

  • IP 地址和埠號是否正確：確保 MySQL 伺服器的 IP 地址和埠號 3306 正確無誤。
  • 防火牆設置：檢查遠端伺服器是否允許來自您本地 IP 地址的連接。
  • MySQL 使用者權限：確保 lily_Admin 帳戶有權從遠端 IP 地址進行連接。

這樣就可以成功連接並開始管理 logisticsdb 資料庫了。
