🧩 二、每一層在做什麼（很重要）

1️⃣ Controller（對外 API）

👉 只做一件事：接 request + 回 response

[HttpPost]
public IActionResult AddPackage([FromBody] Package p)
{
    var service = new PackageService();

    service.AddPackage(p);

    return Ok("OK");
}

❌ 不要寫 SQL
❌ 不要寫 DB code

2️⃣ Service（商業邏輯層）🔥

👉 這一層是「大腦」

public class PackageService
{
    private PackageDAO dao = new PackageDAO();

    public void AddPackage(Package p)
    {
        if (string.IsNullOrWhiteSpace(p.TrackingNumber))
            throw new Exception("TrackingNumber 不可空");

        dao.InsertPackage(p);
    }
}

👉 你可以在這裡加：

  • 驗證
  • 規則
  • 流程控制

3️⃣ DAO（資料存取）🔥🔥

👉 只負責「跟 MySQL 講話」

public class PackageDAO
{
    private string connStr = "Server=192.168.0.101;Database=logisticsdb;User ID=lily_Admin;Password=1234;";

    public void InsertPackage(Package p)
    {
        using (var conn = new MySqlConnection(connStr))
        {
            conn.Open();

            var sql = @"INSERT INTO packages 
                        (TrackingNumber, Sender, Recipient, Status)
                        VALUES (@TrackingNumber, @Sender, @Recipient, @Status)";

            var cmd = new MySqlCommand(sql, conn);

            cmd.Parameters.AddWithValue("@TrackingNumber", p.TrackingNumber);
            cmd.Parameters.AddWithValue("@Sender", p.Sender);
            cmd.Parameters.AddWithValue("@Recipient", p.Recipient);
            cmd.Parameters.AddWithValue("@Status", p.Status);

            cmd.ExecuteNonQuery();
        }
    }
}

4️⃣ Model（資料結構）

👉 你已經有了 👍

public class Package
{
    public string TrackingNumber { get; set; }
    public string Sender { get; set; }

    public string Status { get; set; }
}

5️⃣ Config（進階但重要）

👉 把 connection string 抽出來

public static class DbConfig

}

DAO 改：

private string connStr = DbConfig.ConnStr;
